# Student Management System (C)

## Overview

Student Management System is a console-based application developed in C that allows administrators and students to manage and access student records through separate login portals.

The project was created as a learning exercise to apply fundamental C programming concepts such as structures, functions, file handling, arrays, and user authentication in a practical application.

---

## Features

### Admin Portal

Administrators can:

* Add student records
* View all student information
* Search students using USN
* Access complete student details

### Student Portal

Students can:

* View student records
* Search students by USN
* Access limited student information
